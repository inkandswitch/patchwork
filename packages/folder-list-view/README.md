# Folder List View

A simple list view tool for folders.

Deployed as a dynamic tool in Patchwork.

Build: `pnpm build`

Push to automerge: `pnpm push`

## To make your own tool

- Copy this folder
- Update the code to do what you want
- Remove the Automerge URL in the `push` script; run `pnpm push`, then put the URL you get into the command so it'll keep pushing to the same doc
